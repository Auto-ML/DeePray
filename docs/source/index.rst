Welcome to DeePray's documentation!
===================================

**DeePray** (`深度祈祷`) is a recommendation deep learning platform based on TensorFlow 2
that helps engineers/researchers to build end-to-end pipelines for neural network model training for tablue data tasks.
It is designed to provide Modular, Scalable, Configurable, Easy-to-Use and Extend toolkit for
minimizing developing cost of RS deep neural modeling.
